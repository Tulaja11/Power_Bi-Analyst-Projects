# 📊 Financial Performance & Variance Analysis

A data analytics project that compares **Budget vs Actual** spending across departments to identify variances, overspending, and savings opportunities.

---

## 🛠️ Tools & Technologies

| Tool | Purpose |
|------|---------|
| Python (Pandas) | Data cleaning & variance calculation |
| MySQL | Analytical queries & KPI calculations |
| Power BI | Interactive dashboard & visualizations |

---

## 🎯 Objective

Analyze organizational financial performance to detect:
- Overspending areas by department and category
- Savings opportunities
- Budget utilization trends
- Monthly spending spikes

---

## 📁 Project Files

| File | Description |
|------|-------------|
| `financial_variance_analysis.csv` | Source dataset |
| `Finance_variance.ipynb` | Python data cleaning & EDA |
| `finance_variance.sql` | SQL queries for KPI analysis |
| `finance_variance_analysis.pbix` | Power BI dashboard |
| `Financial Variance Analysis.pdf` | Project report |

---

## 🔄 Workflow

```
Python (Cleaning) → SQL (Analysis) → Power BI (Dashboard)
```

---

## 💡 Key Insights

- Certain departments consistently exceed their budgets
- A small number of cost categories drive the majority of spending
- Spending spikes are concentrated in specific months
- Variance tracking enables more effective cost control
